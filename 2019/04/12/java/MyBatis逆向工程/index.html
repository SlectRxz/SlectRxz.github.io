MyBatis逆向工程的实现。

逆向工程是干嘛的？
我的理解是：逆向工程是根据数据库的表帮助我们直接生成pojo以及mapper中的接口以及关系映射文件。

那么我们应该如何修改配置文件？
在generatorConfig.xml中配置mapper生成的详细信息，注意改下几点：
1.添加要生成的数据库表
2、po文件所在包路径
