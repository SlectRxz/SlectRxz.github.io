    SpringMVC整合Mybatis

    思路：
    Dao层

    • SqlConfig.xml
    • applicationContext-dao.xml
      • 数据库连接池
      • SqlSessionFactroy对象
      • 配置mapper包扫描

    Service层

    • applicationContext-service.xml包扫描其，扫描@Service的注解
    • applicationContext-trans.xml配置事务

    Controller层

    • springmvc.xml
      • 包扫描器，扫描@Controller注解
      • 配置注解驱动
      • 配置视图解析器
      • 配置全局异常处理器
      • 配置多媒体处理器
      • 配置拦截器

    Web.xml

    • 配置Spring容量监听器（找到spring文件）

    • 配置前端控制器（springmvc）
